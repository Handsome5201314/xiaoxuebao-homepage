# 小雪宝 Web UI 交互前端

小雪宝公开前端项目，基于 React、TypeScript 和 Vite 构建，用于展示小雪宝能力介绍、对话体验入口、用户登录、历史记录和管理端原型界面。

## 本地开发

```bash
npm install
npm run dev
```

默认开发服务监听 `127.0.0.1:5173`。

## 构建

```bash
npm run build
```

构建产物输出到 `dist/`，该目录不会提交到仓库。

## 后端网关

`web-gateway/` 包含一个 FastAPI 网关示例，用于连接小雪宝对话服务、用户登录、历史记录和管理端接口。运行前请通过本地环境变量或本地 `.env` 配置必要的服务地址和密钥，密钥文件不要提交到仓库。
